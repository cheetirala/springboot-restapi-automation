# API automation suite with Spring boot, RestAassured, Cucumber and Java

## Overview 
This repository contains an API automation framework written in Java using:
- Spring boot Test with Cucumber BDD
- RestAssured for REST API testing
- Structured Step definitions and reusable clients

## Scenarios Covered
- Create, Read, Update and Delete (CRUD)
- Edge cases: Retrieval of Deleted items

## Setup
- Java 24
- Maven 3.9+

## Run locally
mvn clean test
